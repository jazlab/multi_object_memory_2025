# Task Visualization

This directory contains tools to visualize the task and behavior of the monkeys.
There are two entrypoints:

- `demo_replay_trial.ipynb`. Run this notebook to generate a gif file replaying
  a trial of task and behavior. Here's what that gif looks like:

  <img src="gifs/Perle_2022-06-01_trial_200.gif" width="50%">
- `demo_replay_trials_with_neurons.ipynb`. Run this notebook to generate a video
  of trials of task and behavior with audio showing spike times from some
  neurons. Click on this video to see what that looks like:

[![videos_with_neurons/Perle_2022-05-31.mp4](https://img.youtube.com/vi/MbDBzNsw81M/0.jpg)](https://www.youtube.com/watch?v=MbDBzNsw81M)
